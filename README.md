GIT ref: 
--------

Upon initialization(first time using git for a project)
$ git init
$ git clone <url>

Adding changes
$ git add . 
Commit changes
$ git commit 
Push changes to everyone
$ git push 

Get the updated version
$ git pull origin master

Undo your changes on a file
$ git checkout -- <file>
